# 1) Initiating git

git init #initiate git

git commit -m ‘메시지’ #메세지라는 이름으로 커밋

touch a.txt  #make a new text file

git stautus #상태 확인

git add . #모든 파일을 add

git log # 커밋 히스토리 조회

# 3) CLI 명령어

- pwd : print working directory

- ls : list(파일 목록)

- touch : 파일 만들기

- rm : 파일 삭제

- mkdir: make directory

- cd: change directory 하위 폴더에 들어가기

- cd ..: 상위 폴더로 올라가기

- cd 폴더명 : 해당 폴더로 들어가기


**중요
1. 작업을 하고
2. 변경된 파일을 모아(add)
3. 버전으로 남긴다 (commit)
